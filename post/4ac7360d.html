Golang之匿名函数和闭包

学习 golang 匿名函数和闭包

匿名函数

函数内部没有办法声明有名字的函数就是匿名函数

// 匿名函数

var f1 = func(x, y int) {
	fmt.Println(x + y)
}

func main() {
	f1(1, 2)
	//函数内部没有办法声明有名字的函数
	f1 := func(x, y int) {
		fmt.Println(x + y)
	}
	f1(10, 10)

	//如果只是调用一次的函数, 还可以简写成立即执行函数
	func(x, y int) {
		fmt.Println(x + y)
		fmt.Println("Hello world")
	}(100, 100)
}

闭包(closure)

因为在 golang 中函数是可以作为参数使用的, 当一个函数的参数需要一个无参数的函数时, 我们想传一个有参的函数进去此时就要用到闭包.

// 闭包

//f1 函数要求的参数是无参的函数
func f1(f func()) {
	fmt.Println("this is f1")
	f()
}

//我想将f2传进去, 此时会报错, 所以我们需要闭包
func f2(x, y int) {
	fmt.Println("this is f2")
	fmt.Println(x + y)
}

//此时定义一个f3函数, 对f2进行包装, f3返回的是无参函数, 正好满足f1的需求
func f3(f func(int, int), m, n int) func() {
	temp := func() {
		f(m, n)
	}
	fmt.Println("this is f3")
	return temp
}


func main() {
	f := f3(f2, 100, 200)
	f1(f)
}
