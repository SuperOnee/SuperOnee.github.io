StreamAPI 简介

Java8 两个比较大的改变, 一个是 lambda 表达式, 另一个就是今天的主角 StreamAPI. StremAPI 主要用于处理集合, 它可以对集合进行非常复杂的操作, 例如: 筛选, 分组, 处理数据等.

StreamAPI 给我们操作集合带来了强大的功能, 同时呢上手还非常的容易. 在学习 Stream api 之前呢我们需要学习 Java Lambda 表达式!

lambda 表达式

lambda 表达式就是匿名函数, 对于一个 java 变量我们可以给其赋一个 “值”.

那么如果想把 “一块代码” 赋给一个 java 变量应该怎么做呢? 在 java 8 之前这个是做不到的, 但是 java8 问世之后呢, 利用 lambda 特性就能够做到了. 接下来看一个简单的例子

  1. 创建一个接口, 此接口必须只能有一个 public 方法
public interface NumberOperation {
    int result(int n1, int n2);
}
  1. 实现接口
  • 正常方式实现接口:
@Test
    void lambdaTest() {
        NumberOperation numberOperation = new NumberOperation() {
            @Override
            public int result(int n1, int n2) {
                return n1 + n2;
            }
        };
        int result = numberOperation.result(1, 2);
        System.err.println(result);
    }

    //输出3
  • lamdba: 因为此接口只有一个函数, 那么我们就可以使用匿名函数的方法去实现此接口
@Test
    void lambdaTest() {
        NumberOperation numberOperation = (n1, n2) -> n1 + n2;
        int result = numberOperation.result(1, 2);
        System.err.println(result);
    }
    //输出3

函数引用

函数引用是由 lamdba 演变而来, 当我们 lambda 的参数和返回值与我们需要引用的函数相同的时候, 此时我们就可以使用函数引用, 在 java 中函数引用用双冒号表示, 就像这样: Integer::sum, 直接上代码:

@Test
    void lambdaTest() {
       //lambda的参数为两个int, 返回值为 int1 + int2, 结果为一个int
       //此时呢, 我们就可以使用函数引用来实现我们的方法.
        NumberOperation numberOperation = Integer::sum; //函数引用
        int result = numberOperation.result(1, 2);
        System.err.println(result);
    }
    //输出3

除了函数引用之外呢, 还有一种构造引用, 其实二者没什么区别. 上代码看例子, 我们将使用 java 的 supplier 接口来完成这个例子.

  • java 内置 Supplier 接口
/**
 * Represents a supplier of results.
 *
 * <p>There is no requirement that a new or distinct result be returned each
 * time the supplier is invoked.
 *
 * <p>This is a <a href="package-summary.html">functional interface</a>
 * whose functional method is {@link #get()}.
 *
 * @param <T> the type of results supplied by this supplier
 *
 * @since 1.8
 */
@FunctionalInterface
public interface Supplier<T> {

    /**
     * Gets a result.
     *
     * @return a result
     */
    T get(); //泛型为我们当前匿名函数的返回值, suuplier接口的匿名函数为无参数有返回值的方法, 此时我们就可以调用构造引用来构建一个Person对象
}
  • 例子
@Test
   void lambdaReference() {
       Supplier<Person> supplier = Person::new;
       Person person = supplier.get(); //返回了person对象
   }

Stream 的操作步骤

  1. 创建 Stream: 从一个集合或者数组中获取 Stream.

  2. 中间操作: 对集合/数组进行数据的处理, 得到想要的结果.

  3. 终止操作: 在处理完结果后, 拿到我们需要的数据并返回.

举个简单的例子: 假设我们有一个 Person class, 和一个 Person List
现在有一个需求, 就是找到单身的人并且输出 Person 的姓名.

  1. 创建 PersonClass

  1. peek 操作: peek 是用来处理数据的, 我们可以将流中的元素进行遍历处理
@Test
    void peekTest() {
       //使用无限流Generate生成3个person对象
        List<Person> personList = Stream.generate(Person::new).limit(3).
        collect(Collectors.toList()); //使用Collectors将流收集成List
        //peek 操作将流中的属性改变
        personList.stream().peek(s -> {
            s.setName("peekTest");
            s.setAge(19);
            s.setSingle(false);
        }).forEach(System.err::println);
    }

        //2. 手动实现comparator.compare函数

        //那么倒序怎么做呢? 其实很简单, 上代码
        System.err.println("倒序排列: ");
        //1. 调用reversed方法将其反转
        System.err.println("reverse方法");
        list.stream().sorted(Comparator.comparingInt(Person::getAge).reversed()).forEach(System.err::println);
        //2. 加 - 号
        System.err.println("comparingInt");
        list.stream().sorted(Comparator.comparingInt(s -> -s.getAge())).forEach(System.err::println);
        //一个是手动实现comparator接口 加 - 号
        System.err.println("comparator.compare");
        list.stream().sorted(((o1, o2) -> -o1.getAge() - o2.getAge())).forEach(System.err::println);

        //输出结果:
        //正序排列: comparingInt函数

@Test
    void toMapSolutionTest() {
        List<Person> personList = new ArrayList<>();
        Person michale = new Person("Michale", 19, true);
        Person lily = new Person("Lily", 20, false);
        Person frankly = new Person("Frankly", 21, true);
        Person joe = new Person("Joe", 20, false);
        //这里我们添加一个Joe2进行测试, person的name属性值是一样的都是joe
        Person joe2 = new Person("Joe", 22, true);
        personList.add(michale);
        personList.add(lily);
        personList.add(frankly);
        personList.add(joe);
        personList.add(joe2);

        Map<String, Person> map = personList.stream()
                .collect(Collectors.toMap(Person::getName, (Person person) -> person));
        System.err.println(map);
    }

//终端报错java.lang.IllegalStateException: Duplicate key Person{name='Joe', age=20, single=false}

在上面那个例子中, 我们 key 值发生了重复并没有处理, 所以运行起来就直接抛异常了. 异常信息告诉我们 key 值发生了重复, 接下来就来看看这个问题怎么解决吧, 其实很简单, toMap 提供了对应的解决方案.

@Test
    void toMapSolutionTest() {
        List<Person> personList = new ArrayList<>();
        Person michale = new Person("Michale", 19, true);
        Person lily = new Person("Lily", 20, false);
        Person frankly = new Person("Frankly", 21, true);
        Person joe = new Person("Joe", 20, false);
        //这里我们添加一个Joe2进行测试, person的name属性值是一样的都是joe
        Person joe2 = new Person("Joe", 22, true);
        personList.add(michale);
        personList.add(lily);
        personList.add(frankly);
        personList.add(joe);
        personList.add(joe2);

        Map<String, Person> map = personList.stream()
                //我们只需要实现ToMap的mergeFunction即可解决, 这里replacement为第二个参数
                //第一个参数为原有的值, 第二个为冲突的新的, 这里我们选了replacement就是将原有的替换掉了
                //如果需要保留原来的只需要返回第一个参数即可: (prePerson, replacement) -> prePerson
                .collect(Collectors
                        .toMap(
                                Person::getName, (Person person) -> person, (prePerson, replacement) -> replacement
                        )
                );
        System.err.println(map);
    }

    //输出: {Joe=Person{name='Joe', age=22, single=true}, Frankly=Person{name='Frankly', age=21, single=true}, Michale=Person{name='Michale', age=19, single=true}, Lily=Person{name='Lily', age=20, single=false}}

    //toMap函数还提供了mapSupplier, 我们可以通过mapSupplier接口返回LinkedHashMap

    Map<String, Person> map2 = personList.stream()
                //我们只需要实现ToMap的mergeFunction即可解决, 这里replacement为第二个参数
                //第一个参数为原有的值, 第二个为冲突的新的, 这里我们选了replacement就是将原有的替换掉了
                //如果需要保留原来的只需要返回第一个参数即可: (prePerson, replacement) -> prePerson
                .collect(Collectors
                        .toMap(
                                Person::getName, (Person person) -> person, (prePerson, replacement) -> replacement, LinkedHashMap::new
                        )
                );
        System.err.println(map2);
  1. groupingBy 操作: 我们可以通过 groupingBy 操作对集合进行分组. 上代码.
@Test
    void groupingByTest() {
        List<Person> personList = new ArrayList<>();
        Person michale = new Person("Michale", 19, true);
        Person lily = new Person("Lily", 20, false);
        Person frankly = new Person("Frankly", 21, true);
        Person joe = new Person("Joe", 20, false);
        personList.add(michale);
        personList.add(lily);
        personList.add(frankly);
        personList.add(joe);

        //这里我们通过groupingBy把单身的分为一组, 不单身的分为一组
        Map<Boolean, List<Person>> personGroup = personList.stream()
                .collect(Collectors.groupingBy(Person::getSingle, Collectors.toList()));
        System.err.println(personGroup);

        //输出: {false=[Person{name='Lily', age=20, single=false}, Person{name='Joe', age=20, single=false}], true=[Person{name='Michale', age=19, single=true}, Person{name='Frankly', age=21, single=true}]}

        //同时groupingBy也提供了MapFactory, 我们也可以通过实现这个接口返回LinkedHashMap对象
        Map<Boolean, List<Person>> personGroup2 = personList.stream()
                .collect(Collectors.groupingBy(Person::getSingle, LinkedHashMap::new, Collectors.toList()));
        System.err.println(personGroup2);
        //输出: {true=[Person{name='Michale', age=19, single=true}, Person{name='Frankly', age=21, single=true}], false=[Person{name='Lily', age=20, single=false}, Person{name='Joe', age=20, single=false}]}
        //使用mapping来处理groupingBy后的数据, 这里我们在downstream中传入collectors.mapping, 只取Person的name属性
        Map<Boolean, List<String>> personGroup3 = personList.stream()
                .collect(
                        Collectors.groupingBy(
                                Person::getSingle, LinkedHashMap::new, Collectors.mapping(Person::getName, Collectors.toList())
                        )
                );
        System.err.println(personGroup3);
        //输出: {true=[Michale, Frankly], false=[Lily, Joe]}
    }

  1. NicknameGenerator class
public class NicknameGenerator {

    private static final char [] chars = {'a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j', 'k', 'l', 'm', 'n', 'o', 'p', 'q', 'r', 's', 't', 'u', 'v', 'w', 'x', 'y', 'z'};

    public static String getNickname() {
        StringBuilder builder = new StringBuilder();
        Random random = new Random();
        int loopTimes = random.nextInt(5) + 1;
        for (int i = 0; i < loopTimes; i++) {
            int ranNum = random.nextInt(chars.length);
            builder.append(chars[ranNum]);
        }
        return builder.toString();
    }

    public static String getRandomEmail(int length) {
        StringBuilder builder = new StringBuilder();
        Random random = new Random();
        for (int i = 0; i < length; i++) {
            int ranNum = random.nextInt(chars.length);
            builder.append(chars[ranNum]);
        }
        builder.append("@gmail.com");
        return builder.toString();
    }
  1. 程序入口: HelloWorld.java
public class HelloWorld {
    public static void main(String[] args) {
        List<GroupUser> groupUserList = getGroupUsers();
        Map<Integer, LinkedHashMap<Integer, GroupUser>> mergeResult = groupUserList.stream()
                .collect(
                        Collectors.groupingBy(
                                GroupUser::getGroupId, Collectors.toMap(
                                        GroupUser::getUserId, (GroupUser gp) -> gp,
                                        (gp, replacement) -> replacement,
                                        LinkedHashMap::new
                                )
                        )
                );
        //merged result
        mergeResult.entrySet().forEach(System.err::println);
        //split result
        mergeResult.forEach((key, value)
                -> value.forEach((key1, value1) ->
                System.err.println("key: " + key1 + ", value: " + value1)));
    }

    public static List<GroupUser> getGroupUsers() {
        List<GroupUser> list = new ArrayList<>();
        int num = 1;
        for (int i = 0; i < 3; i++) {
            for (int j = 0; j < 3; j++) {
                GroupUser groupUser = new GroupUser(num, (int) (Math.random() * 100000000));
                list.add(groupUser);
            }
            num++;
        }
        return list;
    }
}
